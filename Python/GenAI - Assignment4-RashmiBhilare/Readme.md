## Assignment 4 — File Handling

Welcome — this folder holds seven short exercises (Task1 through Task7) that practice basic file I/O in Python. Below each task is described in plain language: what it does, what it asks from you (if anything), and a simple command to run it.

### Task 1 — Write sales records
- Summary: Writes a hard-coded list of sales to `sales_data.txt` (one per line), reads and prints that file, and also writes a comma-separated line to `sales_data.csv`.
- Input: none.
- How to Run: 
From "GenAI - Assignment4-RashmiBhilare" folder run command - python "Task1.py"

### Task 2 — Read file in different ways
- Summary: Demonstrates three reading methods (`.read()`, `.readline()`, `.readlines()`) on `sales_data.txt` and shows converting the lines to integers.
- Input: none (reads the file created by Task1 or an existing `sales_data.txt`).
- How to Run: 
From "GenAI - Assignment4-RashmiBhilare" folder run command - python "Task2.py"

### Task 3 — Append new sales
- Summary: Appends three new sales amounts to `sales_data.txt`, prints the updated file, and reports the total number of lines.
- Input: none.
- How to Run: 
From "GenAI - Assignment4-RashmiBhilare" folder run command - python "Task3.py"

### Task 4 — Generate a summary report
